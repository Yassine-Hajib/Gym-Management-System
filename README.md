Gym Management System Platform

A full-stack Gym Management System built with React and Laravel and Mysql .
This platform is designed to help gym owners efficiently manage their gym operations while providing dedicated spaces for coaches and members.


Features Overview

The system is divided into three main roles:
Owner (Admin)
Coach
Member
Each role has specific permissions and dashboards
